# D&D Sheet Link для Owlbear Rodeo

Расширение связывает токены Owlbear Rodeo с персонажами из Firebase-комнаты вашего листка.

## Установка

После публикации проекта на GitHub Pages добавьте в Owlbear Rodeo URL manifest:

```text
https://<ваш-логин>.github.io/<репозиторий>/obr-sheet-link/manifest.json
```

## Как пользоваться

1. Откройте комнату Owlbear Rodeo и добавьте расширение.
2. В панели расширения введите название комнаты из листка персонажа.
3. Выберите токен на сцене Owlbear.
4. Выберите персонажа в списке расширения.
5. Нажмите `Привязать`.

После этого расширение будет читать `hpCur`, `hpMax` и `ac` из листка и показывать локальный оверлей рядом с токеном.

## Важно

Синхронизация односторонняя: листок -> Owlbear. ХП меняется в листке персонажа, а токен обновляет отображение.
